CLI 2.x vs CLI 3.x

- 명령어

  -2.x : vue init '프로젝트 템플릿 이름 ' '파일 위치'

  -3.x : vue create '프로젝트 이름'

- 웹팩 설정 파일

  -2.x : 노출 O ( webpack.config.js 가 존재)

  -3.x : 노출 x (복잡하기 때문에 라이브러리로 관리해줌)

- 프로젝트 구성

  -2.x : 깃헙의 템플릿 다운로드 (github에 있는 내용을 그대로 다운받아서 사용 )

  -3.x : 플러그인 기반으로 기능 추가

  (npm install 로 설치 , 원하는 feature들을 추가가능, 유연하게 설치)

- ES 6 이해도

  -2.x : 필요 X

  -3.x : 필요 O
